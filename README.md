## Installation

1. Clone repository
2. Extract from Zip
3. Run npm install

## Usage 

1. Open command prompt to root file
2. Open MySQL shell
3. Run source db/schema.sql
4. Run quit
5. Run node seeds/index.js
6. Run node server.js
7. Access through Insomnia Core
